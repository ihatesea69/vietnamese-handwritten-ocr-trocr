# Artifacts

Thư mục này dành cho model, checkpoint và các output cục bộ không nên đẩy lên GitHub.

- `models/best_model/`: model đã fine-tune để web app sử dụng.
- `training/`: metadata hoặc checkpoint phụ trong quá trình train.

Repo chỉ giữ lại phần cấu trúc; nội dung nặng bên trong sẽ được `.gitignore` bỏ qua.
